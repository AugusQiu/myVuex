# 基于webpack5.x配置一个简单的vue脚手架
## 相关命令
````
npm init -y
````
````
npm i -D webpack webpack-cli
````
````
npm i vue -S
npm i vue-loader css-loader vue-template-compiler -D
````
````
npm i babel-core babel-loader babel-preset-env -D 
````
````
npm i html-webpack-plugin -D
````
# myVuex
````
npm run dev
````
